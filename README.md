# 🎮 Streamer Overlay Hub

配信画面（OBS）をよりスマートに、自分好みにカスタマイズできる配信者向けの統合オーバーレイツールです！
ゲームの勝敗数や、アビリティ使用回数・デス数などのサブカウンターをリアルタイムで画面に表示・操作できます。

👉 **[ツールのURLを開く](https://【あなたのアカウント名】.github.io/【リポジトリ名】/)**

---

## ✨ 主な機能

- **リアルタイム勝敗カウンター**: 配信中にボタンをポチポチ押すだけで、画面のWIN/LOSS数が変わります。
- **自由なカラーカスタマイズ**: ゲームのUIや自分のチャンネルの雰囲気に合わせて、文字色や影の色を自由に変更可能。
- **マルチレイアウト**: 画面の配置に合わせて「横並び」「縦並び」をワンクリックで切り替えられます。
- **汎用サブカウンター**: デス数やアビリティ回数、視聴者参加型の待機人数など、名前を自由に変えてカウントできます。

---

## 🚀 使い方（OBSへの設定方法）

1. **ツールのURLを開く**: [Streamer Overlay Hub](https://【あなたのアカウント名】.github.io/【リポジトリ名】/) にアクセスします。
2. **URLをコピー**: サイト下部にある「OBS用ブラウザソースURL」の横にある**コピー**ボタンを押します。
3. **OBSにソースを追加**: OBSを開き、ソース一覧の「+」から**「ブラウザ」**を追加します。
4. **URLの貼り付け**: コピーしたURLをURL欄に貼り付けます。
   - **おすすめサイズ**: 幅 `650` / 高さ `800` (操作画面も含むサイズです)
5. **配信中の操作方法**: OBS上のソースを右クリックし、**「相互作用（インタラクト）」**を選ぶと、配信画面上で直接ボタンを押して数字や色を操作できます！
   - ※配信用スマホや別モニターでツールを開いて操作することも可能です。
